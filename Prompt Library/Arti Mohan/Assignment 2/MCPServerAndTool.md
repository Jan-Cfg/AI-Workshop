
## MCP Server
GitHub MCP Server allows AI clients to interact directly with GitHub repositories, pull requests, issues, and workflows. 

## Tool
GitHub acts as the code repository and PR automation tool. The AI client can fetch code, analyze changes, and post review comments.

## Sample Purpose
- Connects to GitHub MCP Server using PAT
- Fetches changed Java files from a PR
- Demonstrates sending code to AI for Clean Code + Security Review
- Shows posting structured review comments
