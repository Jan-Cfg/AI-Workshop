# Learn RAG

The following repository contains code to access an LLM with a RAG server as an intermediary.

## Models
1. LLM: gemma3
2. EMBEDDING: nomic-embed-text

## Tools Used

1. Chainlit: This is a chatbot framework that allows us to create interactive chat applications. It provides a simple interface for building conversational agents with very pleasing UI.
2. Ollama: Open source LLM server that allows us to host and serve our language models. Gemma3 and nomic-embed-text are hosted on Ollama local server.
3. ChromaDB: A vector database that allows us to store and query high-dimensional vectors efficiently. We use it to store the embeddings generated by the nomic-embed-text model.
4. Langchain: A framework used in this case, primarily to help with splitting documents into chunks and uploading them to the Chroma VectorDB
